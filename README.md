# doran-doran
도란도란 가족들과 이야기하기를 바라며 만든 Family Link 대체 앱.

---

## 기능
AI 보호자 인증(얼굴인식 기반)
규칙 설정 및 앱 실행 방지

---

## 사용 방법
앱 설치 시, 보호자 인증을 받는다
보호자가 규칙을 지정한다.
앱 삭제/규칙 번경을 할 때에도 보호자 인증을 받는다.

---

## Language
안드로이드는 Webviewer으로 html 파일 띄워서 Ui 표시.
파일 내부에 있는 에이전트가 html의 명령을 토대로 제어.
